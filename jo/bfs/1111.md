# 1111 설계서

## BFS 조건

1. 시작점
   - 외곽지점
     - 0, N+1
2. 인접
    - 상하좌우
3. 종료점
   - 더이상 큐가 없을때
     - 최솟값을 더이상 발견하지 못할때
     - 정상지점 도착하면 인큐 하지 않음
4. 큐
   - 좌표 정보(x, y)
5. 방문 표시
   - Cost MAP cost 인큐 조건으로 대체

## 실제 코드조건

1. 필요한 변수
   - Cost MAP
     - type: int
   - 입력 변수
     - 도착 좌표 저장 변수
       - int 2개
         - dr, dc
     - N: 맵 크기
       - type: int
     - 높이 저장 MAP
       - type: int
     - 노드
       - 방문할 좌표 저장 X,Y
     - 큐
       - 요소: NODE
       - 크기: MAX_N^2 10
     - 상대 방향 저장용 변수(상하좌우)
       - dR, dC
2. 초기화 방법
   - Cost MAP 초기화
     - 0x7FFFFFFF
   - 나머진 전역 0 초기화
3. BFS 동작
   1. 시작점(테두리)들에 대해 모두 인큐
   2. 탐색 동작 실행
      1. 디큐
      2. 디큐 대상이 산정상이라면 continue
      3. 산 정상이 아니라면 상하좌우 탐색
         1. 경계 검사 (1~N)
         2. COST 비용을 검사해서 이전보다 작으면
            1. COST MAP 갱신
            2. 해당 좌표 인큐
   3. 더이상 큐가 없으면 산 정상 좌표 COST 값 리턴
